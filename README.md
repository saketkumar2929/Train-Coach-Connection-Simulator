This project is a menu-driven C program that simulates train coach connections using a Doubly Linked List.

## Features
- Add coach at front
- Add coach at end
- Remove coach by name
- Display train from front to back
- Display train from back to front
- Create default train layout

## Default Train Layout
D1 -> D2 -> S1 -> S2 -> S3 -> B1 -> B2 -> C1 -> A1 -> H1

## Data Structure Used
- Doubly Linked List

## Menu Options
1. Add Coach at Front  
2. Add Coach at End  
3. Remove Coach  
4. Display Train (Front to Back)  
5. Display Train (Back to Front)  
6. Create Default Train Layout  
7. Exit  

## How to Run
```bash
gcc train_coach_simulator.c -o train
./train
